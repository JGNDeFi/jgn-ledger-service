(JVM) NodeRPCConnection(@Value("${config.rpc.host}") host: String, @Value("${config.rpc.username}") username: String, @Value("${config.rpc.password}") password: String, @Value("${config.rpc.port}") rpcPort: Int)

Wraps an RPC connection to a Corda node.

The RPC connection is configured using command line arguments.

Parameters

host - The host of the node we are connecting to.

rpcPort - The RPC port of the node we are connecting to.

username - The username for logging into the RPC client.

password - The password for logging into the RPC client.
